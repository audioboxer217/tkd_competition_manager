# TKD Competition Manager — Usage Guide

Welcome to the usage documentation for the **TKD Competition Manager**.

This documentation is split by role so each team member gets only the steps they need during a live tournament.

---

## 1) Choose your guide

| Role | Guide |
|------|-------|
| Tournament Administrators | [Administrator Guide](./USAGE_GUIDE_ADMIN.md) |
| Ring / Table Staff (Scorekeepers) | [Ring/Table Staff Guide](./USAGE_GUIDE_RING_TABLE_STAFF.md) |
| General Event Volunteers | [Volunteer Guide](./USAGE_GUIDE_VOLUNTEERS.md) |

---

## 2) Who should use which guide

- **Administrator** — You configure rings and divisions, load competitor rosters, generate brackets, assign rings, and handle live data corrections. Your primary screen is the **Admin Dashboard** (`/admin`).
- **Ring/Table Staff** — You run live matches and record results at your assigned ring. Your primary screen is the **Scorekeeper Console** (`/ring/<ring_id>/scorekeeper`).
- **Volunteer** — You support athlete check-in, staging, and flow near rings. You typically do not enter match results directly.
